## O Que Ja Conhecia


##### Criar um novo repositório
```
git init

```

##### obtenha um repositório
```
git clone <caminhoRepositorio>
```

##### Adicionar Um Arquivo Especifico
```
git add <arquivo>

```

#####  Adicionar Tudo que a de novo no repositorio local

```
git add .
git add *

```
